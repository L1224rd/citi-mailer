Simplest Mailer on Earth!

How to use it:

Send a POST request to https://smoe.herokuapp.com with the following body:

{
host: String, // smtp host, default: 'smtp.gmail.com'
port: Number, // default: 465
secure: Boolean, // true for 465, false for other ports, default: true
user: String, // user email
password: String, // we do nothing with your password
receiver: String, // can be a list of emails separated by comas
subject: String, // subject of the email
body: String, // accepts html
}

The response will be:

// success
{
type: 'success'
info: info
}

// error
{
type: 'error'
info: error
}
